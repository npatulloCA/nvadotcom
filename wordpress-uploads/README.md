# WordPress images

Images downloaded from the New Vintage Amplifiers WordPress export (`newvintageamplifiers.WordPress.2026-03-13.xml`).

- **217 images** (~151 MB)
- Paths mirror the original WordPress uploads structure (e.g. `2013/05/`, `2014/01/`) plus any referenced in page content (e.g. `_img/`).
- To re-run or fetch any new URLs from the XML: `python3 download_wordpress_images.py` from the repo root.
